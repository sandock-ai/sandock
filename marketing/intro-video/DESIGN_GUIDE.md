## 🎨 通用设计规范

### 画布尺寸
- **分辨率**: 1920x1080 (16:9)
- **安全区域**: 留出 5% 边距（96px）

### 字体
- **主标题**: Inter Bold / Poppins Bold, 48-64px
- **副标题**: Inter Medium, 24-32px
- **正文**: Inter Regular, 20-24px
- **代码**: Fira Code / JetBrains Mono, 14-16px

### 配色基础
- **主色调**: 深色背景（#0a0e27, #0f172a）
- **强调色**: 蓝色（#3B82F6）、绿色（#10B981）、紫色（#8B5CF6）、橙色（#F59E0B）
- **文字**: 白色（#FFFFFF）、浅灰（#9CA3AF）

### 图标风格
- **风格**: 线性图标（Line Icons）或扁平图标（Flat Icons）
- **大小**: 48-64px（主要图标）、24-32px（辅助图标）
- **来源**: Heroicons, Lucide Icons, 或自定义

## 设计图 1: Hook 开场 (0:00-0:08)

### 画面布局
```
┌─────────────────────────────────────────┐
│                                         │
│    🤖                    📦             │
│  AI Agent  ────────▶   [Sandbox]       │
│   (左侧)              (右侧发光)        │
│                      </> print()        │
│                      function()         │
│                                         │
│  Give your AI Agent a safe playground  │
│         to run any code.                │
└─────────────────────────────────────────┘
```

### 设计元素详解

#### 1. AI Agent 机器人图标（左侧 1/3）
- **位置**: 画面左侧，垂直居中
- **样式**: 简洁的机器人图标
  - 圆形头部
  - 两个圆点眼睛
  - 简单的天线（可选）
  - 极简风格，线条清晰
- **颜色**: `#5B8DEF`（蓝色）或 `#8B5CF6`（紫色）
- **大小**: 占画面高度的 30-40%（约 300-400px）
- **效果**: 可选微妙的发光效果

#### 2. 箭头（中间）
- **位置**: 从 AI Agent 指向沙箱
- **样式**: 实线箭头，箭头头部清晰
- **颜色**: `#60A5FA`（浅蓝色）或 `#FFFFFF`（白色，50% 透明度）
- **粗细**: 3-4px
- **长度**: 约 200-300px

#### 3. 沙箱容器（右侧 1/3）
- **位置**: 画面右侧，垂直居中
- **样式**: 
  - 圆角矩形（圆角 16-24px）
  - 或立方体（3D 效果，可选）
  - 半透明填充
  - 边缘发光效果（2-3px 光晕）
- **颜色**: 
  - 边框: `#10B981`（绿色）或 `#3B82F6`（蓝色）
  - 填充: `rgba(16, 185, 129, 0.1)`（半透明绿）
  - 光晕: 同边框色，模糊 10-15px
- **大小**: 占画面高度的 40-50%（约 400-500px）

#### 4. 代码符号（沙箱内部）
- **位置**: 沙箱容器内，随机分布或垂直排列
- **符号**:
  - `</>` 符号（顶部）
  - `print()` 文字（中间）
  - `function()` 文字（底部）
- **字体**: Fira Code 或 JetBrains Mono
- **大小**: 18-24px
- **颜色**: `#34D399`（浅绿）或 `#FFFFFF`（白色）
- **效果**: 可选轻微漂浮动画（上下移动 5-10px）

#### 5. 文字叠加（底部）
- **位置**: 画面底部，水平居中，距离底边 80-100px
- **文字**: "Give your AI Agent a safe playground to run any code."
- **字体**: Inter Regular
- **大小**: 24-28px
- **颜色**: `#FFFFFF`（白色）
- **行距**: 1.5
- **对齐**: 居中
- **背景**: 可选半透明深色背景条（`rgba(0, 0, 0, 0.5)`），内边距 16px

#### 6. 背景
- **样式**: 深色渐变
- **颜色**: 从上到下
  - 顶部: `#1e1b4b`（深蓝）
  - 中间: `#0a0e27`（深蓝黑）
  - 底部: `#0f172a`（深黑）
- **纹理**: 可选微妙的网格纹理（10% 透明度）

### 设计要点
- ✅ 保持简洁，不要过度设计
- ✅ AI Agent 和沙箱要清晰可辨
- ✅ 箭头要明显，表示"进入"的动作
- ✅ 沙箱发光效果要明显，传达"安全"的感觉
- ✅ 文字要清晰可读，即使在小屏幕上

  - 颜色: `#9CA3AF`（浅灰）

- **位置**: 画面底部，水平居中，距离底部 60-80px
- **文字**:
  - 第一行: "Persistent Volumes."
  - 第二行: "Your data never disappears."

#### 6. 背景
- **颜色**: `#0f172a`（深蓝黑）
- **渐变**: 可选从左到右的微妙渐变
  - 左侧: `#0f172a`（深蓝黑）
  - 右侧: `#1e1b4b`（深蓝）

- ✅ 容器用虚线，Volume 用实线，对比"临时"vs"持久"
- ✅ Volume 要比容器更醒目（更大、发光）
- ✅ 数据流动要清晰，箭头方向明确
- ✅ 文件图标要多样化（不同类型的文件）
- ✅ 状态说明要有对比（灰色 vs 绿色）
- ✅ 整体传达"数据永不丢失"的概念
